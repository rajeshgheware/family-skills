How to use this guide

Read slowly. Practice deliberately.

i.

One a week is enough

Ten skills, ten weeks. Read one, sit with it, try one practice. Speed defeats the point.

ii.

Pick the practice, not the principle

Reading about resilience does not build resilience. The exercises at the end of each chapter do.

iii.

Return to it

This is not a book you finish. It is a reference you grow into. The same paragraph at fifteen and twenty-five reads differently.
